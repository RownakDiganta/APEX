# Overview

This directory contains the official legal and compliance documentation published by Hack The Box.

These documents define the legal, ethical, privacy, acceptable-use, community, and contractual rules governing use of the Hack The Box platform.

The PolicyAdvisor retrieves information from these documents to determine whether a planned action complies with Hack The Box policies before execution.

# Intended Usage

These documents are used by the APEX PolicyAdvisor to:

- Validate whether an action is within the authorized scope.
- Verify compliance with Hack The Box Terms of Service.
- Check Acceptable Use Policy restrictions.
- Validate publication and disclosure rules.
- Enforce AI and machine learning usage restrictions.
- Reference privacy and security obligations.
- Support policy-aware autonomous penetration testing.

---

# Maintenance

The documents in this directory are official Hack The Box publications and should not be modified manually.

When Hack The Box updates a policy:

1. Download the updated official document.
2. Replace the corresponding PDF.
3. Regenerate the compiled policy database.
